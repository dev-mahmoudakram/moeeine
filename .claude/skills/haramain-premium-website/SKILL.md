# Haramain Premium Website Skill

## Purpose

Use this skill when working on the Laravel 13 premium website for the Haramain mobility service.

This skill guides Claude to design and build a luxury Arabic-first website using Laravel, Blade, Bootstrap, SCSS, localization, subtle animations, and Laravel Boost.

## Project Identity

The project is a premium Saudi mobility service for visitors of the Two Holy Mosques.

The service provides:

- Electric carts
- Wheelchairs
- Trained companions
- Booking services
- Hotel pickup and delivery
- Bus station and parking support
- Support for elderly visitors
- Support for people with disabilities
- Operations center
- Service points
- Fleet tracking
- Maintenance
- Charging
- Reports and dashboards

The website should present the service as:

> A luxury smart mobility and spiritual care platform for serving Guests of Allah with dignity, comfort, and operational excellence.

## Core Positioning

Do not position the service as a simple cart rental website.

Position it as:

- A premium service platform
- A smart operations system
- A dignified mobility solution
- A Saudi hospitality experience
- A service designed for elderly visitors and people with disabilities
- A project suitable for official presentations, partners, hotels, and investors

Core message:

> We do not only provide a cart. We provide a dignified, safe, and organized mobility experience for the Guests of Allah.

## Technical Stack

Always use:

- Laravel 13
- Blade
- Bootstrap 5
- SCSS
- Vite
- Laravel localization
- Arabic-first RTL layout
- English as secondary language
- Vanilla JS for interactions
- CSS animations

Do not use:

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Inertia.js unless already required by the project
- Heavy frontend frameworks

## Required Website Sections

### Hero Section

Purpose:
Create a strong premium first impression.

Should include:

- Premium headline
- Short emotional description
- CTA buttons
- Dark emerald visual style
- Gold accents
- Haram-inspired arch or geometric motif
- Image/visual area
- Subtle hero glow animation

Suggested CTAs:

- احجز خدمتك الآن
- استكشف الخدمات

### Trust Section

Show:

- مركز عمليات 24/7
- مرافقون مدرّبون
- حجز إلكتروني
- عربات كهربائية
- نقاط خدمة
- تتبع وتشغيل ذكي

### About Section

Explain:

- Who the service helps
- Why it exists
- How it improves the visitor journey
- Dignity, privacy, comfort, and independence

### Services Section

Required service cards:

- عربة كهربائية بدون مرافق
- عربة مع مرافق مدرّب
- خدمة الفنادق
- خدمة محطات الباصات والمواقف
- توصيل داخل المنطقة المركزية
- توصيل خارج المنطقة المركزية
- دعم كبار السن وذوي الهمم

Each card should include:

- Title
- Description
- Icon or motif
- CTA
- Hover effect

### Electric Cart Showcase

Present the cart as a premium product.

Mention:

- Foldable
- Easy to store
- Suitable for crowded paths
- Stable and safe
- Battery suitable for daily journeys
- Trackable and maintainable

### How It Works

Use these steps:

1. اختر الخدمة
2. حدد التاريخ والوقت
3. اختر موقع الاستلام والتوصيل
4. أضف بيانات المستفيد
5. أكّد الحجز والدفع
6. استلم الخدمة وتابعها بسهولة

## Visual Style

The website should feel:

- Premium
- Luxury
- Calm
- Trustworthy
- Saudi-inspired
- Spiritual
- Human
- Operationally professional

Use:

- Dark emerald backgrounds
- Deep teal sections
- Luxury gold accents
- Warm ivory text
- Muted mint highlights
- Islamic geometric patterns
- Haram-inspired arches
- Premium cards
- Glassmorphism
- Large spacing
- Subtle animations
- Elegant section dividers

Avoid:

- Generic SaaS look
- Medical/hospital look
- Delivery app look
- Overly colorful UI
- Crowded layouts
- Aggressive animation
- Stock-template feeling
